Detailed Description

template<int W, int K, int S = QUEUESIZE>
class RanSched< W, K, S >

Schedule lookup requests in a random pipeline.

In a random pipeline, a search request jump over all the pipestages to perform the IP lookup task.

Parameters
Wat most W search steps for a request
Knumber of pipe stages
Ssize of request queue
