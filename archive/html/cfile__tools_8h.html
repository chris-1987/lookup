cfile_tools.h A small library containing tools for dealing with files that are compressed with different compressors or not compressed at all. The idea is to recognize the compression automagically and transparently. Files can be opened for reading or writing, but not both. Reading and writing is by different function classes. Access is sequential only.

Support for gzip added by Bernhard Tellenbach bernh.nosp@m.ard..nosp@m.telle.nosp@m.nbac.nosp@m.h@gma.nosp@m.il.c.nosp@m.om

Function prefixes are: cfr_ = compressed file read

Supported: Reading:

  • type recognition from file name extension
  • standard input (filename: '-' )
  • no compression
  • bzip2
  • gzip
