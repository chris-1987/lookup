Detailed Description

template<int W>
struct FNode2< W >

A node in a fixed-stride tree after leaf-pushing.

A node in a leaf-pushed fixed-stride tree.

Each node contains _entrynum entries, where each entry only contains one field shared by child pointer and nexthop+length.

Parameters
W32 or 128 for IPv4 or IPv6, respectively.

Each node contains a set of entries and each entry stores a child pointer or a length/nexthop pair.

Parameters
W32 or 128 for IPv4 or IPv6, respectively.
