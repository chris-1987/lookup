Detailed Description

Definition of a lookup index based on fixed-stride tree.

Copyright (c) 2016, Sun Yat-sen University, All right reserved.

The lookup index consists of three parts: a fast lookup table that contains route information for prefixes shorter than U bits, a table that consists of 2^U entries and each entry stores a pointer to the root of a fixed-stride tree, and a set of fixed-stride trees.
