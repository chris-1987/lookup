Detailed Description

template<int W>
class BTree< W >

Build and update binary tree.

All the prefixes of length $k$ are located in the $k$-th level of the binary tree. A search start from the root node and branch to the higher levels by checking bits in the given IP address.

Parameters
W32 for IPV4 and 128 for IPV6
