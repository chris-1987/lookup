Detailed Description

template<int W, int K, size_t MP = 2 * K + 1, size_t MC = static_cast<size_t>(pow(2, K))>
struct PNode< W, K, MP, MC >::PrefixEntry

Prefix entry.

Each entry consists of three items, which are prefix, length of prefix and nexthop.

A primary node consists of multiple prefix esntries, each entry records a prefix, the length of the prefix and its nexthop. A primary node also contains multiple child pointers.
